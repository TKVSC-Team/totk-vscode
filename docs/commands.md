`Set Up Python Environment` - Sets up plugin, runs automatically on first launch

`Select Python (from detected installs)` - Searches the device for valid Python installations to choose from

`Browse for python.exe` - Prompts the user to select a python executable from their device

`Set Game Dump Path` - Prompts the user to select the romfs folder of their game dump (does not need to be named romfs)

`Rebuild Game Dump Search Index` - Generates an SQLite database of files, archives, and sub-files/archives in the supplied romfs dump

`Rebuild Canonical Path Index` - Generates an SQLite database of canonical paths for files and subfiles in the supplied romfs dump

`Add workspace folder to Projects` - Adds the folder the user has open in VSCode to the Projects tab

`Refresh Projects` - Reloads the projects window view

`Remove from Projects` - Removes the selected Project folder from the Projects tab

`Refresh Game Dump` - Refreshes the Game Dump view

`File Icon Theme Help` - Triggers a popup explaining how to prevent TKVSC icons from overwriting other extension icons

`Choose External Tool for Current File` - Prompts the user to select an external tool to open files matching the extension of the currently selected filetype. Best for unsupported formats

`Open with Associated External Tool` - Opens the currently selected file in a previously assocated external tool

`Add to Project…`- Prompts the user to select one of their Project folders to add the selected file(s) to

`Delete` - Prompts the user to confirm deletion of the selected file(s)

`Rename` - Prompts the user to rename the selected file

`New File` - Creates an empty file using the provided name and extension

`New Folder` - Creates a new folder at the current level

`Copy` - Copies the selected file(s) to clipboard

`Cut` - Copies the selected file(s) to clipboard and deletes them upon Paste

`Paste` - Pastes the file(s) from the clipboard

`Export...` - Prompts the user to select a location to export the file to

`Export Current File to YAML` - Export current file as a `.YAML`

`Open File` - Opens the file in the editor view, if supported.
